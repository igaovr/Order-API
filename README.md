• Order API – Node.js + Express + MongoDB

Esta API foi desenvolvida como parte de um desafio técnico.
O objetivo é criar um serviço simples para gerenciar pedidos, permitindo:

Criar pedidos

Buscar pedido por ID

Atualizar pedido

Excluir pedido

Listar todos os pedidos

Todos os dados são armazenados em um banco MongoDB, usando Mongoose.

• Tecnologias Utilizadas

Node.js

Express

MongoDB / Mongoose

Dotenv

Nodemon (dev)
